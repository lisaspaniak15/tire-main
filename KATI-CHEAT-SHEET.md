# Kati's Complete Setup & Editing Guide

## Part 1: Do This Once — Getting Set Up

These steps only need to be done one time ever.

**Step 1: Install Claude Code**
1. On your Mac, open Safari or Chrome
2. Go to **claude.ai/code**
3. Sign in with your team Claude account
4. Follow the instructions to download and install it

**Step 2: Create a GitHub account**
1. Go to **github.com**
2. Click **Sign Up**
3. Create a free account
4. Send Lisa your GitHub username so she can invite you to the projects

**Step 3: Accept Lisa's invitations**
1. You will get emails from GitHub saying you've been invited
2. Click the link in each email to accept
3. Do this for all 4 invitations

**Step 4: Download the projects to your computer**

Once Lisa has invited you, open Claude Code and say each of these one at a time:

- *"Clone the project at https://github.com/lisaspaniak15/tire-main into /Users/katispaniak/TIRE Projects"*
- *"Clone the project at https://github.com/lisaspaniak15/tire-clients into /Users/katispaniak/TIRE Projects"*
- *"Clone the project at https://github.com/lisaspaniak15/tire-agents into /Users/katispaniak/TIRE Projects"*
- *"Clone the project at https://github.com/lisaspaniak15/truth-in-real-estate into /Users/katispaniak/TIRE Projects"*

This downloads everything to your computer. You only do this once.

## Part 2: Every Time You Start Working

Do these steps at the beginning of every editing session.

**Step 1: Open Claude Code**
Open the Claude Code app on your Mac just like you would open any other app.

**Step 2: Tell Claude which project you want to work on**

Say one of these depending on what you want to edit that day:

| If you want to edit... | Say this to Claude |
|---|---|
| The main website, Ask Kati, FAQ, Privacy Policy, or Seller's Playbook | *"Open the project at /Users/katispaniak/TIRE Projects/tire-main"* |
| The Seller & Buyer portal | *"Open the project at /Users/katispaniak/TIRE Projects/tire-clients"* |
| The Agent portal | *"Open the project at /Users/katispaniak/TIRE Projects/tire-agents"* |
| The CRM (coordinate with Lisa first) | *"Open the project at /Users/katispaniak/TIRE Projects/truth-in-real-estate"* |

**Step 3: Get Lisa's latest changes**

Say: *"Pull the latest changes from Lisa"*

This makes sure you have any updates Lisa made since you last worked.

## Part 3: Editing a Page

Tell Claude exactly what you want to edit and it will open the right file and pull it up in your browser automatically.

| What you want to edit | Say this to Claude |
|---|---|
| The main Truth in Real Estate homepage | *"Open and run the main homepage on port 3001 so I can edit it"* |
| The Ask Kati page | *"Open and run the Ask Kati page on port 3001 so I can edit it"* |
| The FAQ Video Library page | *"Open and run the FAQ video library page on port 3001 so I can edit it"* |
| The Privacy Policy page | *"Open and run the Privacy Policy page on port 3001 so I can edit it"* |
| The Seller's Playbook page | *"Open and run the Seller's Playbook page on port 3001 so I can edit it"* |
| The Seller & Buyer portal | *"Open and run the Seller and Buyer portal on port 3002 so I can edit it"* |
| The Agent portal | *"Open and run the Agent portal on port 3003 so I can edit it"* |
| The CRM | *"Open and run the CRM on port 3004 so I can edit it"* |

Once the browser opens, just tell Claude in plain English what you want to change. For example:
- *"Make the headline bigger"*
- *"Change the button color to gold"*
- *"Add a new section below the hero with three columns"*

Claude will make the change and you will see it update in your browser.

## Part 4: Saving Your Work

When you are happy with your changes, say:

*"Commit and push my changes"*

Claude will save everything and send it to Lisa automatically.
