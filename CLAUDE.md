## What this project is

This is the **public marketing website** for Truth in Real Estate. It will eventually live at **https://truthinrealestate.com**. This is the main brand/marketing site for Kati's business.

**This project is not yet fully built out.** It is planned for a future phase as part of the rebrand from katispaniak.com to truthinrealestate.com.

**This is NOT the CRM.** The CRM (internal tool) is a separate repo called `truth-in-real-estate`.
